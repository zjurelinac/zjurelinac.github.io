Projects

NoteStack

An advanced note-taking web application built with Python Flask and AngularJS. Features organizing notes into categories and by tags, as well as a simple markup language for rich content.

View the app – not yet online

Pljuska

A command-line interpreter for a custom Hash scripting language, written entirely in Haskell. Developed as a project for a Programming in Haskell course on FER Zagreb.

Project page

Grupa Kemija

A web site for chemistry group in my highschool – XV. Gimnazija, featuring custom content management system and Q&A, written in PHP framework CodeIgniter and jQuery.

Link to website

Poly

A C++ library which extends the language with the capability of representing and calulating with polynomials. Includes basic arithmetic operations, differentiation, factorization and root finding algorithms.

Github repo

Linker

A Python web app for group sharing of links. Written in Flask and AngularJS, it enables people to easily share links with their friends and coworkers.
