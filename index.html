Projects

NoteStack

An advanced note-taking web application built with Python Flask and AngularJS. Features organizing notes into categories and by tags, as well as a simple markup language for rich content.

View the app – not yet online

Pljuska

A command-line interpreter for a custom Hash scripting language, written entirely in Haskell. Developed as a project for a Programming in Haskell course on FER Zagreb.

Project page

Grupa Kemija

A web site for chemistry group in my highschool – XV. Gimnazija, featuring custom content management system and Q&A, written in PHP framework CodeIgniter and jQuery.

Link to website

Poly

A C++ library which extends the language with the capability of representing and calulating with polynomials. Includes basic arithmetic operations, differentiation, factorization and root finding algorithms.

Github repo

Linker

A Python web app for group sharing of links. Written in Bottle framework and AngularJS, it enables people to easily share links with their friends and coworkers.

Github repo

Interests

Programming

I like making new things, and programming gives me the best opportunity to do just that.

I started while still in primary school, since then tried many, from VHDL and Assembler to Haskell, and learning each was a lot of fun.

Languages I'm familliar with
(and what I did with them):

  • C and C++ — mostly algorithmic programming, but also some higher-level projects in C++
  • Java — GUI applications: Calculator, Notepad, physics simulator, Minesweeper
  • Python — web applications (REST APIs), with Django and Flask
  • Haskell — my favourite programming language, still learning all it's capabilities
  • JavaScript — the easiest one to play with, used it mostly with jQuery or AngularJS as a frontend for one of the web apps
  • PHP — web applications (obviously :), using CodeIgniter framework
  • HTML and CSS/LESS — this very page, as well as some others
